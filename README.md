## Project Overview

The goal of Virago is to raise awareness of disparities in athlete compensation by presenting wage data in a clear, engaging, and easy-to-understand format. The site uses visual comparisons and interactive elements to encourage exploration and understanding of inequality in sports.

This project focuses on **communication, design, and data storytelling**, making complex social issues more approachable to a broad audience.

## Features

- Interactive visual comparisons of athlete compensation
- Clear presentation of wage disparities across sports
- Responsive design for desktop and mobile
- User-focused layout emphasizing clarity and accessibility
